Install
------------------------------------------------------------------------------

``prompt_ide`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install prompt-ide

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade prompt-ide
